## Installation
1. `pip install virtualenv`
2. `virtualenv venv`
3. `source venv/Scripts/activate` or `venv/Scripts/activate`
4. `pipenv install`
5.  Run server `python app.py`

 Server should be running on `http://localhost:3000/`
